# Rainbow

# 환경설정
언어 : JAVA 8

프레임워크 : Spring 4.3.11

서버 : Apache Tomcat 8.5

데이터베이스 : Oracle 11g

기타기술 : JavaScript

운영체제 : MacOS

# 구현기능
- 회원가입 / 로그인
- 카테고리별 상품 목록
- 키워드 검색
- 상품 상세 페이지
- 장바구니
- 주문하기 / 주문 목록
- 상품 후기 / 댓글
- 상품 등록
- 회원 목록
